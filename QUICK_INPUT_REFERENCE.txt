🎯 5-SECOND SUMMARY
═══════════════════════════════════════════════════════════════════════════════

1. Open:     http://localhost:8501
2. Click:    "📋 Submit Application"
3. Modify:   Credit score, income, etc. (or use defaults)
4. Submit:   Click "🚀 Submit Application & Analyze"
5. Wait:     15-30 seconds for analysis
6. Review:   See decision and agent reasoning


📋 FORM FIELDS QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

APPLICANT INFO:
  First Name:           Text (max 50 chars)
  Last Name:            Text (max 50 chars)
  Email:                Email format
  Phone:                Text (e.g., 555-0001)
  Date of Birth:        Calendar picker
  SSN (Last 4):         Numbers only (exactly 4 digits)
  Annual Income:        Numbers in dollars
  Employment Status:    Dropdown (Employed, Self-employed, Retired, Unemployed)
  Years Employed:       Decimal number (0-50)
  Current Employer:     Text

LOAN DETAILS:
  Loan Amount:          Numbers in dollars
  Loan Purpose:         Dropdown (Home, Auto, Personal, Business)
  Loan Term:            Slider (12-360 months)
  Interest Rate:        Decimal percentage (optional)

CREDIT PROFILE:
  Credit Score:         Numbers (300-850)
  Accounts Open:        Numbers (0+)
  Delinquencies (30d):  Numbers (0+)
  Delinquencies (90d):  Numbers (0+)
  Total Debt:           Numbers in dollars
  Recent Inquiries:     Numbers (0+)
  Bankruptcy History:   Dropdown (None, Chapter 7, Chapter 13, etc.)

DOCUMENT VERIFICATION:
  Income Verification:  Dropdown (Verified, Pending, Failed)
  Income Confidence:    Slider (0-100%)
  Employment Verif.:    Dropdown (Verified, Pending, Failed)
  Employment Conf.:     Slider (0-100%)
  Documents Provided:   Multi-select checkboxes

📊 WHAT YOU'LL SEE AFTER SUBMISSION
═══════════════════════════════════════════════════════════════════════════════

RESULT DISPLAY:
  ✅ APPROVED         (Green - Low risk, all checks pass)
  ❌ REJECTED         (Red - High risk, critical issues)
  ⏳ MANUAL_REVIEW   (Orange - Borderline, needs human review)

DECISION CARD:
  • Final Decision (color-coded)
  • Score: 0-100 (Higher = stronger approval)
  • Processing Time: Seconds taken
  • Application ID: Unique identifier

REASONING:
  • 1-2 paragraph explanation from AI

AGENT ANALYSES (Can expand each):
  • Document Verification: Score + detailed analysis
  • Credit Analysis: Score + detailed analysis
  • Risk Assessment: Score + detailed analysis
  • Compliance Check: Score + detailed analysis
  • Decision Agent: Final synthesis

RISK FLAGS:
  • List of identified risk factors

COMPLIANCE ISSUES:
  • List of regulatory concerns (if any)

🔑 KEY FIELDS THAT IMPACT DECISIONS
═══════════════════════════════════════════════════════════════════════════════

MOST IMPORTANT (Strongly impacts decision):
  1. Credit Score         ← Primary factor
  2. Annual Income        ← Affects debt-to-income ratio
  3. Total Debt           ← Combined with income = DTI
  4. Delinquencies        ← Recent problems = red flag
  5. Loan Amount          ← Must be reasonable for income

MODERATE IMPORTANCE:
  6. Years Employed       ← Stability indicator
  7. Verification Status  ← Document authenticity
  8. Recent Inquiries     ← Sign of financial stress
  9. Loan Purpose         ← Type matters
  10. Bankruptcy History  ← Historical problems

LESS IMPACT (Still checked):
  11. Number of accounts  ← Diversity indicator
  12. Employment status   ← Type of work

🐛 COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════════════════════

"Page not loading"
  → Refresh browser (F5)
  → Check: http://localhost:8501 is correct

"Form fields disabled/grayed out"
  → Page loading
  → Wait a moment and try again

"Submit button doesn't work"
  → Fill all required fields (marked *)
  → Try again

"Application takes too long"
  → Normal! Takes 15-30 seconds
  → Claude API processing all 5 agents

"Don't see results after waiting"
  → Refresh page (F5)
  → Try submitting again

"Can't modify values"
  → Make sure you're on "Submit Application" page
  → Try scrolling to field

"Results show error"
  → Check internet connection
  → Check API key is valid
  → Check servers are running

🎓 TIP: HOW DECISIONS ARE MADE
═══════════════════════════════════════════════════════════════════════════════

Score Calculation:
  1. Each agent scores 0-100
  2. Scores are averaged
  3. Final score determines decision:
     • Score ≥ 75  → APPROVED ✅
     • Score < 40  → REJECTED ❌
     • 40 ≤ Score < 75 → MANUAL_REVIEW ⏳
